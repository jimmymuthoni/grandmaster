### Instructions to Set up and run this project:

1. clone this repo:

```
git clone https://github.com/jimmymuthoni/grandmaster.git

```

2. Install the dependencies.

```
sudo apt update
sudo apt-install qt6-base-dev build-essential cmake qt6-svg-dev

```

3. Build the project:

```
cd grandmaster
mkdir build
cd build
cmake ..
make

```

4. Run the project:

```
./grandmaster
```
